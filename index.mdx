Spec2Vec is a Python framework that turns any 1D time-series or 2D time-frequency representation into a compact, interpretable "spectral fingerprint" — a vector of information-theoretic, statistical, and spatial metrics that uniquely characterizes a signal. Developed for geophysics and planetary exploration, it provides a transparent, physics-aware alternative to black-box deep learning for signal characterization and clustering. You can apply it to raw waveforms, STFT spectrograms, or continuous wavelet transforms with just a few lines of Python.

## How It Works

<Steps>
  <Step title="Install Spec2Vec">
    Clone the repository and install dependencies into a fresh conda environment with Python 3.10.
  </Step>
  <Step title="Load Your Data">
    Pass your 1D time-series or 2D spectrogram array directly to the `GISQAPipeline` — no special data format required.
  </Step>
  <Step title="Extract Spectral Features">
    Call `compute_pointwise_metrics_from_spec()` to receive a pandas DataFrame of interpretable spectral features ready for clustering, classification, or visualization.
  </Step>
</Steps>
